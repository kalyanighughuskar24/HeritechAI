# HeritechAI

HeritechAI is an AI-powered web platform developed to create a smart and interactive community where users can upload, manage, and explore AI-related content. The system provides secure authentication, admin management, OTP verification, and content approval functionalities to ensure a safe and organized environment for users.

This project is developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP as the local server environment.

🔗 **Live Demo:** [heritechai.infinityfreeapp.com](https://heritechai.infinityfreeapp.com)

# 🌟 Project Overview

The main objective of HeritechAI is to build a centralized platform where users can interact with AI-related resources and administrators can manage the entire system efficiently.

The platform includes:

- User authentication system
- OTP verification
- Admin dashboard
- Content upload and management
- Contact system
- Secure session handling
- Responsive user interface

The project follows a modular structure for easy maintenance and scalability.
